## [1.0.0] - 2025-09-01
### Added
- Complete Electron desktop application
- Multi-source AI content aggregation
- Claude API integration for content analysis
- SQLite local database
- React frontend with modern UI
- Cross-platform builds (macOS, Windows, Linux)
- Background scheduler for automated updates
- Bookmarking and read tracking features

- Reorganized project structure to follow standards
- Improved performance with parallel data collection
